CPXERR_Q_NOT_POS_DEF

5002 Q in '%s' is not positive semi-definite

#define CPXERR_Q_NOT_POS_DEF 5002
The Q matrix associated with the quadratic objective or with a quadratic constraint must be positive semi-definite (for minimizations). Check the appropriate quadratic term(s). If only the objective is quadratic, you can use the solution target parameter to solve models either to local optimality or to global optimality. For more detail about this choice, see solution target type in the manual Parameters of CPLEX.
